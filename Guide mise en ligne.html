Guide complet

Mettre votre site en ligne

Étape par étape, de zéro à votre site publié — avec panneau d'administration pour modifier votre contenu.

1

Créer un compte GitHub

GitHub est l'endroit où votre site est stocké. C'est gratuit et nécessaire pour que le panneau d'administration fonctionne.

→ Créer un compte GitHub
Conseil
Utilisez la même adresse e-mail que pour Netlify. Choisissez un nom simple, par ex. louisemaselis.
2

Créer un nouveau dépôt (repository)

Une fois connectée à GitHub, créez un dépôt — c'est le dossier en ligne qui contiendra tous les fichiers de votre site.

Sur GitHub → bouton vert "New"
  • Repository name : louisemaselis-coaching
  • Visibility : cochez Public
  • Laissez tout le reste par défaut
  • Cliquez "Create repository"
3

Uploader vos fichiers sur GitHub

Dans votre nouveau dépôt vide, cliquez sur "uploading an existing file" (lien bleu au centre de la page).

Glissez-déposez tous les fichiers du dossier netlify que vous avez téléchargé (le ZIP décompressé). Incluez bien les sous-dossiers admin/ et content/.

Fichiers à uploader
  • index.html — votre site FR
  • nl.html — votre site NL
  • site.css — les styles
  • cabinet-v3.jpg — la photo du cabinet
  • image-slot.js — script photo
  • admin/index.html + admin/config.yml
  • content/temoignages-fr.json, temoignages-nl.json, contact.json
  • _redirects

En bas de page, cliquez "Commit changes" pour sauvegarder.

4

Créer un compte Netlify et connecter GitHub

Netlify héberge votre site gratuitement et le remet à jour automatiquement à chaque modification.

→ Créer un compte Netlify
Étapes sur Netlify
  • Connectez-vous avec votre compte GitHub (bouton GitHub)
  • Cliquez "Add new site" → "Import an existing project"
  • Choisissez GitHub → autorisez Netlify à accéder à vos repos
  • Sélectionnez louisemaselis-coaching
  • Laissez les paramètres par défaut → cliquez "Deploy"

Netlify génère une URL provisoire comme random-words.netlify.app — votre site est déjà en ligne !

5

Activer Netlify Identity

C'est ce qui protège votre panneau d'administration avec un mot de passe.

Dans Netlify → votre site → onglet "Integrations"
  • Cherchez "Identity" et cliquez "Enable Identity"
  • Dans les paramètres Identity → Registration → choisissez "Invite only" (seule vous pouvez créer un compte)
  • Cliquez "Invite users" → entrez votre adresse e-mail
  • Vous recevrez un e-mail d'invitation — cliquez le lien et créez votre mot de passe
6

Activer Git Gateway

Git Gateway permet au panneau d'administration d'enregistrer vos modifications directement dans GitHub.

Dans Netlify → votre site → "Integrations"
  • Cherchez "Git Gateway" → cliquez "Enable Git Gateway"
  • C'est tout — ça prend 10 secondes !
7

Connecter votre nom de domaine

Achetez d'abord votre domaine (si ce n'est pas encore fait) sur Namecheap ou Infomaniak (~12€/an pour un .be).

→ Infomaniak (recommandé, belge)
Dans Netlify → "Domain management" → "Add a domain"
  • Tapez louisemaseliscoaching.be → cliquez "Verify"
  • Netlify vous donne 2 nameservers (ex. dns1.p01.nsone.net)
  • Sur votre registrar (Infomaniak) → Domaines → DNS → remplacez les nameservers par ceux de Netlify
  • Propagation : 2 à 24 heures maximum
  • Netlify active automatiquement le HTTPS (cadenas vert)
8

Accéder au panneau d'administration

Une fois le site en ligne, rendez-vous sur :

https://louisemaseliscoaching.be/admin
Ce que vous pouvez modifier
  • Témoignages FR & NL — ajouter, modifier, supprimer
  • Informations de contact — e-mail, téléphone, adresse, Instagram
Comment ça marche
Vous modifiez → cliquez "Publish" → Netlify remet votre site à jour en 30 secondes automatiquement.

Récapitulatif des outils utilisés

  • GitHub — stockage des fichiers (gratuit)
  • Netlify — hébergement du site (gratuit)
  • Decap CMS — panneau d'administration à /admin (gratuit)
  • Infomaniak / Namecheap — nom de domaine (~12€/an)
  • Netlify Identity — connexion sécurisée à l'admin (gratuit)
